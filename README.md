# 🌱 EcoRuta BAQ

EcoRuta BAQ es una plataforma web desarrollada como Proyecto Integrador que busca fomentar el reciclaje responsable mediante una guía educativa, consulta de puntos de reciclaje y un sistema de recompensas para incentivar la participación ciudadana.

## 🚀 Funcionalidades del MVP

- Registro de usuarios
- Inicio de sesión
- Guía de reciclaje
- Consulta de puntos de reciclaje
- Registro de entregas
- Carga de evidencia fotográfica
- Validación por administrador
- Sistema de puntos
- Ranking ecológico
- Dashboard administrativo

## 🛠 Tecnologías

### Frontend
- HTML5
- CSS3
- JavaScript (SPA)

### Backend
- Node.js
- Express.js

### Base de datos
- PostgreSQL
