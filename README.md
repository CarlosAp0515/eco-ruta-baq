# 🌱 EcoRuta BAQ

## Descripción

EcoRuta BAQ es una plataforma web desarrollada como Proyecto Integrador que busca promover el reciclaje responsable mediante una guía educativa, consulta de puntos de reciclaje y un sistema de recompensas por reciclar.

La plataforma permitirá a los usuarios aprender cómo reciclar correctamente diferentes residuos, registrar sus entregas, subir evidencia fotográfica y acumular puntos una vez sus reportes sean validados por un administrador.

---

## 🚀 MVP

- Registro de usuarios
- Inicio de sesión
- Guía de reciclaje
- Consulta de puntos de reciclaje
- Registro de entregas
- Evidencia fotográfica
- Validación por administrador
- Sistema de puntos
- Ranking ecológico
- Dashboard administrativo
